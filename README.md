# ML-Stroke-Prediction
This project uses **R** and **tidymodels** to predict stroke risk based on patient data.

### Key Results:
* **Best Model:** Logistic Regression (AUC: 0.841)
* **Performance:** Successfully caught 72% of stroke cases in the test set.
* **Deployment:** Includes a Shiny App for clinical use.

### How to Run:
1. Open the R script in RStudio.
2. Ensure `tidymodels` and `shiny` are installed.
3. Run the code to launch the predictor app.
